PRUEBA

Los siguientes son pasos a seguir para vincular un proyecto en git

Pasos 1

  • Crear carpeta nueva con nombre de proyecto
  • Crear pagina de prueba en VSC u otro IDE
  • Poner dicha pagina en la carpeta con nombre del proyecto
  • Seleccionar la carpeta o abrirla
  • Cick derecho en la carpeta y seleccionar Git bash

Pasos 2

  • Ubicarse en la consola de Git
  • Escribir ls para ver que archivos existen en la carpeta
  • Escribir pwd para ver la ubicacion del repositorio
  • Escribir git init para inicializar el proyecto(de este modo crea un nuevo repositorio de git)

Luego de esto, se tiene que vincular el reposiorio con una cuenta y el nombre de un desarrollador para los fututos commit. Apareceran los datos del autor y fecha de la modificacion para esto, haremos lo siguiente

  • Escribir git config --global user.email example123@gmail.com
  • Escribir git config --global user.name NombreDelAutor

De este modo queda registrado el nombre y mail del desarrollador en el repositorio
